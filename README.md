## 5. –†–∞—Å—á—ë—Ç CLMM –ø–æ–∑–∏—Ü–∏–π (Concentrated Liquidity)

- –ù–∏–∂–Ω—è—è –≥—Ä–∞–Ω–∏—Ü–∞: **-15%** –æ—Ç —Ç–æ—á–∫–∏ –≤—Ö–æ–¥–∞

### –§–æ—Ä–º—É–ª—ã

**Liquidity (L):**
$$L = \frac{V_0}{2\sqrt{P_0} - \frac{P_0}{\sqrt{P_b}} - \sqrt{P_a}}$$

–≥–¥–µ:
- $V_0$ ‚Äî –Ω–∞—á–∞–ª—å–Ω–∞—è —Å—Ç–æ–∏–º–æ—Å—Ç—å –ø–æ–∑–∏—Ü–∏–∏

- $P_a$ ‚Äî –Ω–∏–∂–Ω—è—è –≥—Ä–∞–Ω–∏—Ü–∞ —Ä–µ–Ω–∂–∞
- $P_b$ ‚Äî –≤–µ—Ä—Ö–Ω—è—è –≥—Ä–∞–Ω–∏—Ü–∞ —Ä–µ–Ω–∂–∞

**–ö–æ–ª–∏—á–µ—Å—Ç–≤–æ –∞–∫—Ç–∏–≤–æ–≤ –≤–Ω—É—Ç—Ä–∏ —Ä–µ–Ω–∂–∞ ($P_a < P < P_b$):**
$$x_{BTC} = L \times \left(\frac{1}{\sqrt{P}} - \frac{1}{\sqrt{P_b}}\right)$$

$$y_{USDC} = L \times \left(\sqrt{P} - \sqrt{P_a}\right)$$

**–°—Ç–æ–∏–º–æ—Å—Ç—å –ø–æ–∑–∏—Ü–∏–∏:**

### –°—Ä–µ–¥–Ω—è—è —Ü–µ–Ω–∞ –ø–æ–∫—É–ø–∫–∏ BTC (–ø—Ä–∏ –≤—ã—Ö–æ–¥–µ –∑–∞ —Ä–µ–Ω–∂)
